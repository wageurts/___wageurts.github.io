YAGNI

You aint gonna need it!

Easier to add something

then to delete a part of the interface.

Use the standard verbs

GET, PUT, POST, DELETE

Make sure the service is idempotent

The same request should always return the same result

and should have the same effect on the application.

Use nouns not actions

DO: /offer, /report, /book

DONT: /createOffer, /retrieveReport, /deleteBook
